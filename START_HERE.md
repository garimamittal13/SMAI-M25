# Start Here

If you're reviewing this repository quickly, these are the best places to start.

## Recommended order

### 1. `assignment-1/`
**Why open this first:** strongest structured-data analytics work  
**What it shows:** exploratory data analysis, grouped summaries, heatmaps, sampling studies, k-NN classification, and regularized regression on tabular data.

### 2. `assignment-2/`
**Why open this second:** best unsupervised learning and model diagnostics  
**What it shows:** K-Means clustering, Gaussian Mixture Models, PCA, clustering validation, and preprocessing for improved structure recovery in distorted datasets.

### 3. `assignment-5/`
**Why open this third:** strongest real-world analytics flavor  
**What it shows:** KDE-based analysis, time-series forecasting, anomaly detection, and generative modeling.

## What this repository is meant to demonstrate

This repository is organized by coursework, but the underlying work reflects a broader data-science workflow:
1. understand the dataset  
2. clean and transform the data  
3. build baseline and advanced models  
4. evaluate with appropriate metrics  
5. visualize results and extract insights

## Tech stack used across the repo
Python, NumPy, Pandas, Scikit-learn, PyTorch, Matplotlib, Seaborn, Jupyter Notebook
